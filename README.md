# Polk Russia Bot

## Introduction

The official bot of the Immortal Regiment of Russia.

## Installation

1. Clone repository.
2. Create `.env` file.
3. Install `poetry`.
```console
> pip install poetry
```
4. Install depends.
```console
> poetry install
```
5. Run this project.
```console
> poetry run python -m source
```
